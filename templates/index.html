About AVE

{% include 'ave.html' %} is a text-based adventure game engine. It is written using Python.

You can make your own games for {% include 'ave.html' %} using the {% include 'ave.html' %} language. Once you have written a game, you can upload it to the game library for anyone to play.

You can play {% include 'ave.html' %} games on this website, or download {% include 'ave.html' %} to play in terminal on Linux or MacOS, or in commmand prompt on Windows.
