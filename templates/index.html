About AVE


{% include 'ave.html' %} running on the EMF TiLDA Mkπ badge.
{% include 'ave.html' %} began life as a Python game. At Electromagnetic Field 2016 (EMF) we made a version of {% include 'ave.html' %} that runs on the TiLDA Mkπ badge that was given to every attendee. Shortly after EMF, we began work on a javascript version of {% include 'ave.html' %}, this website and Android and iOS apps (coming soon).

If you want to play AVE on a non-Linux/Mac computer, you can download our VirtualBox image.

Making Your Own Games

If you would like to write your own games for {% include 'ave.html' %}, have a look at these instructions. Once you have written a game, you can submit it to the game library here. You can view all user submitted games in the library. Once a game has been included in the game library, it will become playable on any version of {% include 'ave.html' %} (as long as the player is connected to the internet).
